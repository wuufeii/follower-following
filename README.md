# follower-following
followers与following比较(可看哪些是相互follower)


### 使用方法
<div>1.拉取项目</div>
<div>2.进入项目，安装依赖：npm install</div>
<div>3.运行前端项目: npm run dev</div>
<div>4.进入node目录,启动服务：node index.js</div>
